Visualization of Energy Consumptions (Cambridge Energy Data Lab - Challenge 2)

This notebook will show a series of visualization of the data provided for the Challenge 2.

The following visualizations are proposed:

* 1) A histogram of the distribution of the consumption values; * 2) A scatterplot of all the values provided; * 3) A box plot of the consumption values per day; * 4) A bar chart of the total consumption per day; * 5) A half hourly box plot of the consumption values; * 6) A half hourly bar chart of the total consumption.

Each graph comes with the code snippet used to make it, a brief description and some interpretations.

In the graph above we have the time on the x axis and the energy consumption on the y axis. The colors show the three groups found by the kmeans algorithm. The first group (blue points) contains the consumption values below the mean that can be considered as low consumption values, the second one (purple points) contains the values right above the mean that can be considered medium consumption, the third group (red points) contains the extreme values, which can be considered high. Hovewer, the regular consumption lays between the first two cluesters. (It is to notice that the hole in scatterplot shows that there is missing data for the 4th and 5th of May)

The scatterplot above shows some peaks in the values that suggest a periodicity trought the days of the week. This will be investigated comparing the distribution of the data for each day using a box plot.

# dayly box plot of the energy consumption
from pylab import boxplot
tobox = []
tobox.append([])
oldd = timestamp[0]
daycnt = 0
labels = [timestamp[0].strftime('%a %m/%d')]
for d,v in zip(timestamp,consumption): # arranging the data as required by the boxplot function
    if oldd.day == d.day and oldd.month == d.month:
        tobox[daycnt].append(v)        
    else:
        labels.append(d.strftime('%a %m/%d'))
        daycnt += 1
        tobox.append([])
        oldd = d
figure(figsize=(16, 6))            
boxplot(tobox)
xticks(range(len(labels)),labels,rotation=40)
ylabel('Energy Consumption (W)')
xlabel('Day (each box considers all the samples collected in one day)')
show()

In the graph above every box describes the distribution of the consumption values for a single day. The box extends from the lower to upper quartile values of the data, with a line at the median. The whiskers extend from the box to show the range of the data. Flier points are those past the end of the whiskers (those points are considered outliers).

In this graph we can notice that in a good number of cases the consumptions were, on average, higher during the weekends. Also, during the weekends we can notice the highest number of outliers (very high consumption values). Hoverwer, the periodicity is not clear because for two of weekends considered, the ones located around the days 05/08 and 05/16, we didn't have any high consumptions.

# half hour boxplot
tobox = [list() for _ in range(48)]
for h in range(0,24):
    for d,v in zip(timestamp,consumption):
        if d.hour == h:
            if d.minute < 30:
                tobox[2*h].append(v)
            else:
                tobox[2*h+1].append(v)
figure(figsize=(16, 6))            
boxplot(tobox)
labels = []
for h in range(24):
    labels.append('%d:00-%d:30'%(h,h))
    labels.append('%d:30-%d:59'%(h,h))
xticks(range(0,48),labels,rotation=50)
ylabel('Energy Consumption (W)')
xlabel('Half hour')
show()

Here, each box describes the distribution fo the data in 30 minutes. It is easy to notice that the average consumption is higher between 10:00 and 13:00. The large amount of outliers between 15:30 and 23:00 suggests that the consumption in the afteernoon and the evening is irregular. Also, there is an increase of the consumption during the evening hours (20:00-1:00).
